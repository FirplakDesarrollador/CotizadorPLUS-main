# Cotizador PLUS - Overview

Cotizador PLUS es una aplicación web basada en Next.js.

## Tecnologías Principales
- **Framework:** Next.js
- **Estilos / UI:** (A completar a medida que se explore el repositorio)
- **Despliegue:** Preparado para Vercel o servidores Node.js.

## Estructura de Proyecto
- `src/`: Código fuente de la aplicación (posiblemente contiene `app/`, `components/`, etc.)
- `public/`: Archivos estáticos.
- **Data Docs:** Archivos de Excel (Ej. `25083 - Related Urban Construction...xlsx`, `Simulación muebles CEMA (1).xlsx`) que parecen usarse como base para cotizaciones o simulaciones de precios.
